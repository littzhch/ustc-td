# 局内视觉方案蓝图

这些参考图不是自由绘画，而是可复刻的工程蓝图。每张 SVG 都使用 1180 x 720 坐标系，正式实现时可以把道路、建筑、部署点、怪物和攻击特效迁移成 Canvas、DOM 或 WebGL 数据。

## 共同硬约束

- 地图不是单张底图：道路、建筑、核心、部署点、单位、攻击特效都必须是代码绘制或实例化对象。
- 道路使用多段线数组，怪物沿道路中心线移动；道路宽度用于碰撞和视觉。
- 建筑使用明确矩形或多边形占位，部署点使用固定坐标，不能靠图片猜位置。
- 参考图尺寸固定为 1180 x 720，与当前游戏画布一致。
